About Grade Oven

Grade Oven, written by Mikel Mcdaniel, is a minimalist web server where instructors can make assignments to build/run/grade student code. In practice it's more flexible than that but the main goal is allow students to submit programming assignments and get instant feedback with a focus on security (protecting the server from potentially malicious students).
